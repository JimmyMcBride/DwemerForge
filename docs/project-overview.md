## Local Notes

DwemerForge is planned as an Electron + Svelte + Tailwind desktop app for managing Elder Scrolls Online addons.

The first differentiator is dependency-aware install/update: when a user installs an addon, DwemerForge should resolve required dependencies, skip already installed dependencies, and install missing required dependencies automatically.

The project is currently in brainstorming/planning. No app scaffold has been created yet.
