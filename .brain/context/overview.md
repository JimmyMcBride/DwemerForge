DwemerForge is planned as an ESO addon manager built with Electron, Svelte, and Tailwind.

Primary product goal: install/update ESO addons with automatic required dependency resolution so users do not have to manually chase library downloads.

Initial planning lives in `.plan/brainstorms/dwemerforge-eso-addon-manager-requirements-and-product-shape.md`.
